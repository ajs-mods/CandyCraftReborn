# CandyCraft Reborn

CandyCraft Reborn is a Minecraft Forge mod for Minecraft 1.20.1.

## Development Setup

1. Install Java 17.
2. Open this project in your IDE.
3. Run one of the following tasks:
   - `./gradlew genIntellijRuns` for IntelliJ IDEA
   - `./gradlew genEclipseRuns` for Eclipse
4. Build with:
   - `./gradlew build`
5. Run the dev client with:
   - `./gradlew runClient`

## Project Info

- Minecraft: 1.20.1
- Forge: 47.4.20
- Mod ID: `candycraftreborn`
- Package: `com.ajsmods.candycraftreborn`
