# CandyCraft Reborn

A magical candy-themed Minecraft mod with sweet adventures and confectionery content.

[![Build](https://github.com/ajs-mods/CandyCraftReborn/actions/workflows/build.yml/badge.svg)](https://github.com/ajs-mods/CandyCraftReborn/actions/workflows/build.yml)

## Download & Install

1. Install [Minecraft Forge](https://files.minecraftforge.net/) for **Minecraft 1.20.1**
2. Download the latest `candycraftreborn-*.jar` from the [Releases page](https://github.com/ajs-mods/CandyCraftReborn/releases)
3. Place the `.jar` file in your `.minecraft/mods/` folder
4. Launch Minecraft with the Forge 1.20.1 profile

> **Tip:** You can also grab development builds from the [Actions tab](https://github.com/ajs-mods/CandyCraftReborn/actions) (click a run → Artifacts → candycraftreborn).

## Testing & Feedback

Found a bug or have feedback? [Open an issue](https://github.com/ajs-mods/CandyCraftReborn/issues/new/choose) with:

- Minecraft version & Forge version
- Steps to reproduce the issue
- Screenshots or crash logs if applicable
- Any other mods installed

## Development Setup

1. Install Java 17
2. Clone this repository
3. Open in your IDE and run:
   - `./gradlew genIntellijRuns` (IntelliJ IDEA)
   - `./gradlew genEclipseRuns` (Eclipse)
4. Build: `./gradlew build`
5. Run dev client: `./gradlew runClient`
